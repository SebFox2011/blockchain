# Gestion de soldes via blockchain

Ce dépôt contient une API Node.js avec MongoDB et un client web.

## Démarrage du serveur

```bash
cd server
npm install
npm start
```

## Démarrage du client React

```bash
cd front
npm install
npm run dev
```

L'application frontend est désormais construite avec [Vite](https://vitejs.dev/) et React/TypeScript. L'interface utilise [Material-UI](https://mui.com/) pour la mise en forme.
